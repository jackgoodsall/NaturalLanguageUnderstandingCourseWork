# Solo Rebuild Plan

## Goal

Rebuild the coursework from the official materials alone and determine whether a fully solo implementation could match or beat the current team branch results.

## Milestones

1. Establish clean solo branch inputs.
2. Build a fresh `Solution B` baseline.
3. Build a fresh `Solution C` baseline.
4. Compare solo rebuild results against the team branch.

## Acceptance tests

1. Branch contains only official coursework materials plus empty solo scaffolding.
2. New `Solution B` and `Solution C` code paths are independent of the team implementations.
3. Metrics are recorded in a reproducible way.
